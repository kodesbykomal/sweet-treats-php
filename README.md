# 🍰 Sweet Treats – PHP Web Application

Sweet Treats is a full-stack web application developed using PHP that allows users to explore, manage, and interact with a variety of dessert products. The application provides a clean and responsive interface along with backend functionality for handling product data and user interactions.

---

## 🚀 Features

- 🍩 Display a variety of sweet products with images and details
- 🔍 User-friendly navigation and structured layout
- 🛒 Product listing and basic interaction functionality
- 📱 Responsive design for mobile and desktop
- ⚙️ Backend integration using PHP
- 🗂 Organized project structure for scalability

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

### 1️ Clone the Repository
git clone https://github.com/yourusername/sweet-treats-php.git


### 2️ Move to Server Directory
- For XAMPP → `htdocs/`
- For WAMP → `www/`

- htdocs/Sweet_Treats


### 3️ Start Server
- Start **Apache**
- Start **MySQL**

### 4️ Configure Database (if applicable)
- Open **phpMyAdmin**
- Create a database (e.g., `sweet_treats`)
- Import `.sql` file (if provided)
- Update database credentials in config file

### 5️ Run Application
Open browser:

**http://localhost/Sweet_Treats**
